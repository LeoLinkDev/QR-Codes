## Typography

### Font Family
- Family: [Outfit](https://fonts.google.com/specimen/Outfit)
- Weights: 400 (Regular), 700 (Bold)

### Hierarchy
- **Heading 1**: 22px (`1.375rem`), Weight 700, Line-height 1.2
- **Body Copy**: 15px (`0.9375rem`), Weight 400, Line-height 1.4
- **Small Text**: 11px (`0.6875rem`), Text-align Center

## UI Components

### Cards
- **Background**: White
- **Border Radius**: 20px (`1.25rem`)
- **Padding**: 1rem (`16px`)
- **Max Width**: 320px (`20rem`)

### Buttons (.btn-home)
- **Shape**: Pill (Border Radius `2rem`)
- **Padding**: `0.75rem` vertical, `1.5rem` horizontal
- **Style**: White background, Slate 900 text, Bold

### Interactive States
- **Hover Effect**:
  - Transform: `translateY(-5px)` (Cards) or `translateY(-2px)` (Buttons)
  - Transition: 0.2s ease

## Spacing & Borders

### Border Radius
- **Container**: 20px (`1.25rem`)
- **Inner Image**: 10px (`0.625rem`)
- **Button**: 32px (`2rem`)

### Spacing System
- **Base Unit**: 16px (`1rem`)
- **Card Padding**: 16px (`1rem`)
- **Text Content**: 24px Top/Bottom (`1.5rem`)

## Shadows

| Type | CSS Value | Usage |
|------|-----------|-------|
| **Default** | `0px 25px 25px rgba(0, 0, 0, 0.0477)` | Resting state for cards |
| **Hover** | `0px 32px 32px rgba(0, 0, 0, 0.1)` | Hover state for interactive elements |
| **Button** | `0px 5px 10px rgba(0, 0, 0, 0.05)` | Resting state for buttons |
